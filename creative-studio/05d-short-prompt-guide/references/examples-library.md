## Quick Reference by Shot Type

### Establishing Shots (Wide, Set Scene)
**Cinematic:**
```
Format & style: Epic establishing, cinematic scope
Wide shot of ancient temple at sunrise, fog rolling through columns
Camera: Slow crane up revealing full structure
Lighting: Golden hour, dramatic shadows, mist visible
Audio: Morning ambience, distant chanting, peaceful
```

**Urban:**
```
Format & style: Urban establishing, gritty realism
Wide shot of busy intersection, traffic and pedestrians crossing
Camera: High angle, static, capturing flow
Lighting: Harsh midday sun, high contrast
Audio: Traffic, horns, footsteps, urban cacophony
```

### Close-Ups (Details, Emotion)
**Emotional:**
```
Format & style: Dramatic close-up, intimate emotion
Extreme close-up on eyes, tears welling up
Camera: Static, shallow depth of field
Lighting: Soft natural, catching moisture
Audio: Quiet breathing, ambient silence
```

**Object Detail:**
```
Format & style: Macro product detail, commercial
Extreme close-up of leather texture, stitching visible
Camera: Slow dolly across surface
Lighting: Raking side light revealing texture
Audio: Material sounds subtle, no music
```

### Transitions (Between Scenes)
**Match Cut Setup:**
```
Format & style: Smooth transition, seamless
Close-up of coffee mug being set down on table
Camera: Overhead, static
Lighting: Warm morning light
Audio: Mug sound (match point), ambience shift
```

**Whip Pan Transition:**
```
Format & style: Energetic transition, dynamic
Medium shot, whip pan right ending in motion blur
Camera: Extremely fast pan, handheld
Lighting: Changing during pan (setup for cut)
Audio: Whoosh sound, audio transition
```

## Usage Guide

**Finding the Right Example:**
1. Identify your use case (filler, UGC, product, etc.)
2. Browse relevant category
3. Note format+style pattern
4. Adapt structure to your needs
5. Maintain 8 core components (see great-prompt-anatomy)

**Adapting Examples:**
- Keep format+style declaration
- Swap subject/action for your needs
- Adjust lighting/audio to match
- Reference camera-movements for vocabulary

**Combining Patterns:**
- Mix styles (e.g., "UGC aesthetic with cinematic lighting")
- Blend approaches (descriptive + directive elements)
- Cross-reference categories for inspiration
